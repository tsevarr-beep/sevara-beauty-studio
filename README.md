# Sevara Beauty Studio

Modern ve premium güzellik salonu mobil uygulaması projesi.

## Proje Hakkında
Bu proje, kadınlara yönelik modern bir beauty salon mobil uygulaması olarak tasarlanmıştır.  
Kullanıcıların online randevu oluşturabileceği, hizmetleri inceleyebileceği ve salon ile etkileşim kurabileceği kullanıcı dostu bir sistem geliştirilmiştir.

## Özellikler
- Online randevu sistemi
- Hizmet listeleme
- Uzman profilleri
- Kampanya ekranı
- Kullanıcı profili
- Modern UI/UX tasarımı
- Mobil uyumlu tasarım

## Kullanılan Teknolojiler
- React
- TypeScript
- Vite
- Firebase
- Modern Mobile UI Design

## Amaç
Güzellik salonları için dijital müşteri yönetimini kolaylaştırmak ve modern kullanıcı deneyimi sunmak.

# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/01830c2c-d7cd-494a-91d5-94c90d01f4f1

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
